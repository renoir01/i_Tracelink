## Technical Tips for Recording

### Pre-Recording Checklist

- [ ] Charge device to 100%
- [ ] Clear notifications
- [ ] Disable auto-brightness
- [ ] Set brightness to 100%
- [ ] Turn off auto-rotate
- [ ] Enable Do Not Disturb
- [ ] Close background apps
- [ ] Test audio levels
- [ ] Practice run-through 2-3 times

### Recording Settings

**Screen Recorder App:**
- Resolution: 1080p (1920x1080)
- Frame rate: 30 FPS minimum, 60 FPS preferred
- Bit rate: High quality
- Audio: Enable microphone for narration
- Countdown: 3 seconds before recording

**Device Settings:**
- Enable "Show Touches" in Developer Options
- Disable auto-sleep during recording
- Use portrait orientation (standard mobile)

### Narration Tips

1. **Use a Script:**
   - Write out full narration beforehand
   - Practice reading it smoothly
   - Time yourself to stay within 5 minutes

2. **Voice Quality:**
   - Use external microphone if possible
   - Record in quiet environment
   - Speak clearly and at moderate pace
   - Pause slightly between sections

3. **Enthusiasm:**
   - Sound excited about features
   - Vary your tone to maintain interest
   - Emphasize key points

### Editing Tips

**Video Editing Software:**
- Mobile: InShot, KineMaster, CapCut
- Desktop: DaVinci Resolve (free), Camtasia, Adobe Premiere

**Editing Checklist:**
- [ ] Cut out long loading times
- [ ] Speed up slow animations (1.5x)
- [ ] Add text overlays for key points
- [ ] Include transitions between sections
- [ ] Add background music (low volume, subtle)
- [ ] Color correction for vibrant display
- [ ] Add intro title card (2-3 seconds)
- [ ] Add outro with contact info

**Text Overlays to Add:**
- Feature names as you demonstrate them
- Key metrics (iron content, quality grade)

## Common Mistakes to Avoid

### ‚ùå Don't Do This:

1. **Spending too much time on authentication**
   - "Now I'll type my email... e-x-a-m-p-l-e..."
   - Solution: Use auto-fill or edit out the wait

2. **Showing every single field in forms**
   - Slowly filling every input
   - Solution: Pre-fill or use cut transitions

3. **Waiting for slow loading**
   - Standing still watching a spinner
   - Solution: Speed up video or cut the wait

4. **Clicking around aimlessly**
   - No clear path through features
   - Solution: Follow the script precisely

5. **Poor audio quality**
   - Background noise, echo, mumbling
   - Solution: Use good microphone, quiet space

6. **No narration or unclear narration**
   - Silence or saying "um" repeatedly
   - Solution: Script and practice

7. **Screen too dim or unreadable**
   - Low brightness, small text
   - Solution: Max brightness, zoom when needed

8. **Too many error states**
   - Showing bugs or crashes
   - Solution: Test thoroughly before recording
